# Piwigo-Export-Script
EN: This is a Python3 script that allows you to export all your images with all categories. It then creates a log with the missing images, a finish log and an HTML log file.
DE: Dies ist ein Python3-Skript, mit dem Sie Ihre gesamten Bilder mit allen Kategorien exportieren können. Danach wird ein Protokoll mit den fehlenden Bildern, ein Finish-Log und eine HTML-Protokoll-Datei erstellt.
***************************************************************************
EN: Always be careful, this script does not interfere with the existing installation, but creates a copy of the images from all categories and subfolders (as seen in Piwigo). Since Piwigo only ever uploads the current year/month/day, a backup of the created structure is necessary in order to export the image collection if necessary. Make sure that there is enough disk space available. Also take a look at the screenshots.

I repeat, I give no guarantee of success and use at your own risk. You should know what you are doing!
I would be very happy about a cup of coffee to provide or support other scripts.
-------------------------------------------------------------------------------------------
DE: Immer mit Vorsicht, dieses Skript greift nicht in die bestehende Installation ein, sondern erstellt aus allen Kategorien, sowie aus allen Unterordnern (wie in Piwigo gesehen) eine Kopie der Bilder. Da Piwigo immer nur das aktuelle Jahr/Monat/Tag hochlädt ist ein Backup der angelegten Struktur notwendig, um eventuell die Bildersammlung zu exportieren. Achten Sie darauf, das genügend Speicherplatz zur Verfügung steht. Schauen Sie sich auch die Screenshots an.

Ich wiederhole, ich gebe keine Garantie auf Erfolg und die Nutzung erfolgt auf eigene Gefahr. Sie sollten wissen was Sie tun!
